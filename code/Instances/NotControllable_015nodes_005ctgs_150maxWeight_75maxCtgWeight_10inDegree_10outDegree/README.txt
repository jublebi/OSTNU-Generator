(O)STNU RANDOM INSTANCE BENCHMARK
==============================

This directory contains 0005 NOT dynamic controllable (NOTDC) (O)STNU random instances.
Each instance is built by a random algorithm tuned by the following parameters:
#nodes:				  15
#contingent:		   5
max Outdegree:		  10
max IDegree:		  10
edgeProbability:	0,20
#max weight:		 150
#max contingent weight:	  75
#max contingent range:	  50
#oracles:	   5
